The Algorithm

2-Call Reflect-then-Refine Pipeline

Implements three research algorithms from Pryzant et al. (arXiv:2601.13922) and LangChain's Prompt Optimization work — fused into an efficient 2-call pipeline.

0
Local Scoring 0 API calls

5-dimension heuristic scores clarity, specificity, completeness, token efficiency, and actionability locally. Builds gradient report of deficits.

1
Reflection Pass ~70% of prompts

If avg score < 7.5: Call 1 diagnoses weaknesses and returns a structured scratchpad with coverageScore, leakageRisk, and highestImpactFix. Strong prompts skip this call entirely.

2
Optimization Pass always runs

Call 2 receives: raw prompt + gradient report + dynamic few-shot examples from history + scratchpad + target model tag. Applies the matched framework (RISEN / RTF / TAG / COSTAR) and enforces all 5 essential elements.

✓
Leakage Check + History Save

Local Jaccard heuristic flags shallow rewrites. On accept, the pair is appended to personal history — feeding dynamic examples for future enhancements.
